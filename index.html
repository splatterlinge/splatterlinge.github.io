Welcome to Splatterlinge.

This game is just for fun! Don't take it serious!

Build

Requirements

$ sudo apt-get update
$ sudo apt-get install build-essential cmake
$ sudo apt-get install libopenal-dev libvorbis-dev libglew-dev

Build

$ git clone https://github.com/splatterlinge/Splatterlinge.git
$ cd Splatterlinge
$ git checkout master
$ cmake .
$ make

Run Game

$ cd Splatterlinge
$ ./bin/Splatterlinge
