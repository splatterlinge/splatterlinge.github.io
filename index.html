Splatterlinge

A project of advanced game development at the university of Ravensburg-Weingarten

Welcome to Splatterlinge.

This game is just for fun! Don't take it serious!

Build

Requirements

$ sudo apt-get update
$ sudo apt-get install build-essential cmake libopenal-dev libvorbis-dev libglew-dev

Build

$ git clone https://github.com/zwostein/Ununoctium.git
$ cd Ununoctium
$ git checkout master
$ cmake .
$ make

Run Game

$ cd Ununoctium
$ bin/Ununoctium
